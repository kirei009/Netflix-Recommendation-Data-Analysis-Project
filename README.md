Netflix Recommendation System

Project Overview
This project analyzes Netflix data and provides insights using Python.
It demonstrates basic data analysis and preprocessing techniques commonly
used in machine learning projects.

Objective
- Load and explore Netflix dataset
- Understand movies and TV shows distribution
- Perform basic data analysis using Python

Technologies Used
- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy

Files in This Repository
- `Copy of Untitled0.ipynb` : Jupyter Notebook containing the code
- `netflix_data.csv` : Dataset used for analysis

How to Run the Project
1. Download the repository
2. Open the `.ipynb` file in Google Colab or Jupyter Notebook
3. Make sure `netflix_data.csv` is in the same folder
4. Run all cells

Dataset
The dataset contains information about:
- Movies and TV Shows
- Genre
- Release year
- Duration
- Country

Conclusion
This project helps beginners understand how real-world datasets
are handled and analyzed using Python.
